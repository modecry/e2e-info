# Пример работы cypress + react

## Начало работы

### Установка пакетов для фронта
`cd ./example-client/`

`npm install`

### Установка пакетов для тестов
`cd ./example-tests/`

`npm install`

### Запуск фронта

`cd ./example-client/`

`npm start`

Фронт запускается на `http://localhost:3000/`

### Запуск cypress

`cd ./example-tests/`

`./node_modules/.bin/cypress open`
